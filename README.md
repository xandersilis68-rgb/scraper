# WC 2025 volleyball efficiency scraper

### what it does

- every hour
- github actions runs
- auto discovers ALL MEN players in World Champs
- scrapes 5 efficiencies:
  - serve
  - reception
  - attack
  - block
  - dig
- saves results into `/data`
  - latest csv / json
  - timestamped csv / json

### where to see the table

go to the `/data/` folder in your repo

open the newest `.csv`  
github renders it as a table automatically

### important notes

this is scraping. if volleyballworld changes layout this will need updating.
